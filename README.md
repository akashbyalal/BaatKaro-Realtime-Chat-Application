# BaatKaro - Real-Time Chat Application

A real-time chat application built using React and Firebase.  
Users can create profiles, send text and image messages, and view shared media within conversations.

---

## 🚀 Features

- 🔐 User authentication (Firebase Auth)
- 💬 Real-time messaging (Firestore)
- 🖼️ Image sharing (Firebase Storage)
- 👤 Profile setup (name, bio, avatar)
- 🔍 User search functionality
- 📁 Media sidebar (view shared images)
- ⚡ Instant updates without refresh
- 📱 Responsive UI

---

## 🛠️ Tech Stack

- Frontend: React, CSS
- Backend: Firebase (Auth, Firestore, Storage)
- State Management: Context API
